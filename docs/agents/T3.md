# T3 Agent (Twin Delayed DDPG Baseline)

The **T3** agent is a custom implementation of the baseline Twin Delayed Deep Deterministic Policy Gradient (TD3) algorithm, adjusted for the Overlay Cognitive Radio Network environment.

---

## 1. Algorithmic Overview
T3 is a model-free, online actor-critic algorithm designed for continuous action spaces. It addresses the systematic overestimation bias of classic DDPG through three primary mechanisms:

1.  **Twin Critics**: Maintains two independent Q-value networks ($Q_{\phi_1}, Q_{\phi_2}$). The target update utilizes the minimum of the two predictions to reduce positive bias:
    $$y_t = r_t + \gamma (1 - d_t) \min_{j=1,2} Q_{\phi_{j, target}}(s_{t+1}, a_{t+1})$$
2.  **Delayed Policy Updates**: Updates the actor network $\theta$ and target networks slower than the critics (typically once every two critic steps, controlled by `policy_delay`).
3.  **Target Policy Smoothing**: Adds clipped noise to target actions to prevent overfitting to Q-value function spikes:
    $$a_{t+1} = \text{clip}\left( \pi_{\theta, target}(s_{t+1}) + \epsilon, \, a_{min}, \, a_{max} \right)$$
    $$\epsilon \sim \text{clip}\left( \mathcal{N}(0, \sigma^2), -c, c \right)$$

---

## 2. Environment Interface & Exploration
*   **State Inputs**: Receives the raw 4D observation vector from the environment representing instantaneous channel gains:
    $$s_t = [h_{PT, PR}, \, h_{SUs, SUR}, \, h_{SUR, SUd}, \, h_{SUs, PR}]$$
*   **Action Output**: Maps state directly to a 2D continuous action vector:
    $$a_t = [P_{s1}, \, \beta]$$
    using a fully connected network followed by a Sigmoid layer forcing outputs in $[0, 1]$.
*   **Exploration Noise**: Standard additive Gaussian noise is added during training:
    $$a_{explore} = \text{clip}\left( a_t + \mathcal{N}(0, \sigma_{explore}^2), \, 0.0, \, 1.0 \right)$$

---

## 3. Training & Network Layout
*   **Replay Buffer**: Utilizes a standard Flat Replay Buffer. Transitions are stored as independent tuple slices $(s_t, a_t, r_t, s_{t+1}, d_t)$.
*   **Objective Functions**:
    *   **Critic Loss**: Minimizes Mean Squared Bellman Error:
        $$\mathcal{L}_{critic}(\phi) = \frac{1}{B} \sum \left( Q_\phi(s_t, a_t) - y_t \right)^2$$
    *   **Actor Loss**: Enforces standard scalar policy update to maximize critic evaluations:
        $$\mathcal{L}_{actor}(\theta) = - \frac{1}{B} \sum Q_{\phi_1}(s_t, \pi_\theta(s_t))$$
